# 广西大学 硕士 毕业 学位论文 XeLaTex模板

#### 介绍
导入overleaf中, 要切换为 XeLaTex 编译器.

在mian.tex里撰写论文(建议交给大模型完成); 引用文献放在ref.bib; 西大论文的标准参考在PDF里. 

#### 关键词: 
广西大学, 硕士论文, 毕业论文, 学位论文, LaTex模板, XeLaTex.
