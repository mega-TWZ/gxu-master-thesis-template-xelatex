# 广西大学 硕士 毕业论文 LaTex模板--tangwenzi

### 介绍
将文件夹 "广西大学硕士论文模板" 导入overleaf中, 要切换为 XeLaTex 编译器.

在main.tex里撰写论文, 建议将整个tex传入大模型, 去生成论文; 

引用文献放在ref.bib里; 西大的论文格式放在PDF里;

### 关键词: 
广西大学, 硕士论文, 博士论文, 毕业论文, 学位论文, LaTex模板, XeLaTex编译.
